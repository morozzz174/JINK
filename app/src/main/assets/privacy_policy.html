2. Сбор и использование данных

При регистрации в Приложении пользователь предоставляет следующие персональные данные:

  • Адрес электронной почты (при регистрации по email)
  • Номер телефона (при регистрации по SMS)

Все пароли генерируются локально на устройстве пользователя и могут быть сохранены только в локальное хранилище устройства по выбору пользователя.

3. Цели обработки персональных данных

Персональные данные пользователей используются исключительно для:

  • Идентификации пользователя в приложении;
  • Обеспечения работы учётной записи;
  • Предоставления услуг подписки "Премиум";
  • Связи с пользователем по вопросам работы приложения.

4. Рекламные материалы

Приложение может отображать рекламные материалы, предоставляемые рекламной сетью (Яндекс.Мобильная реклама).

Рекламная сеть может собирать анонимные данные об устройстве для показа релевантной рекламы. Эти данные не идентифицируют пользователя и не связаны с персональными данными.

5. Покупки внутри приложения

Приложение предлагает функцию отключения рекламы через платную подписку (Премиум).

Оплата обрабатывается через платёжную систему (Google Play Billing / RuStore). Приложение не обрабатывает и не хранит данные банковских карт.

6. Хранение данных

Настройки пользователя (длина пароля, типы символов, выбранная тема) хранятся локально на устройстве с использованием Android DataStore.

Сохранённые пароли хранятся в зашифрованном хранилище устройства.

7. Защита данных

Пароли генерируются с использованием криптографически стойкого генератора случайных чисел (SecureRandom).

Аутентификация пользователей осуществляется через Firebase Authentication, который обеспечивает безопасное хранение учётных данных.
