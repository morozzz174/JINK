Политика конфиденциальности

Дата последнего обновления: 25.04.2026

1. Общие положения

Приложение "Генератор паролей" (далее - "Приложение") распространяется бесплатно и предназначено для генерации безопасных паролей на устройстве пользователя.

Настоящая Политика конфиденциальности определяет порядок обработки и защиты персональных данных пользователей Приложения.

2. Сбор и использование данных

Приложение НЕ собирает, не хранит и не передаёт третьим лицам какие-либо персональные данные пользователей.

Все пароли генерируются локально на устройстве пользователя и могут быть сохранены только в локальное хранилище устройства по выбору пользователя.

Приложение не требует регистрации и не запрашивает персональные данные.

3. Рекламные материалы

Приложение может отображать рекламные материалы, предоставляемые рекламной сетью (Яндекс.Мобильная реклама).

Рекламная сеть может собирать анонимные данные об устройстве для показа релевантной рекламы. Эти данные не идентифицируют пользователя и не связаны с персональными данными.

4. Покупки внутри приложения

Приложение предлагает функцию отключения рекламы через платную подписку (Премиум).

Оплата обрабатывается через платёжную систему RuStore. Приложение не обрабатывает и не хранит данные банковских карт.

5. Хранение данных

Все настройки пользователя (длина пароля, типы символов, выбранная тема) хранятся локально на устройстве с использованием Android DataStore.

Сохранённые пароли хранятся в зашифрованном хранилище устройства.

6. Безопасность

Мы применяем стандартные меры безопасности для защиты данных на устройстве пользователя.

Пароли генерируются с использованием криптографически стойкого генератора случайных чисел (SecureRandom).

7. Контакты

По вопросам, связанным с настоящей Политикой конфиденциальности, обращайтесь по адресу: support@example.com

8. Изменения политики

Мы оставляем за собой право изменять настоящую Политику конфиденциальности. Изменения вступают в силу с момента публикации обновлённой версии Приложения.
