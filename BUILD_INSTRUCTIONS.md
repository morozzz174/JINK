# Инструкция по сборке и публикации в RuStore

## 1. Сборка Release APK/AAB

### Предварительные требования
- Android Studio Iguana+ (2024.1+)
- JDK 17+
- Gradle 8.2+

### Создание keystore для подписи
```bash
keytool -genkeypair -v -keystore release.keystore -alias release -keyalg RSA -keysize 2048 -validity 10000 -storepass YOUR_PASSWORD -keypass YOUR_PASSWORD
```

### Настройка переменных окружения
```bash
export RELEASE_STORE_PASSWORD="your_password"
export RELEASE_KEY_ALIAS="release"
export RELEASE_KEY_PASSWORD="your_password"
```

### Сборка debug APK
```bash
./gradlew assembleDebug
# Результат: app/build/outputs/apk/debug/app-debug.apk
```

### Сборка release AAB (для RuStore)
```bash
./gradlew assembleRelease
# Результат: app/build/outputs/apk/release/app-release.aab
```

### Сборка release APK
```bash
./gradlew assembleRelease -PbundleInDebug=false
# Результат: app/build/outputs/apk/release/app-release.apk
```

## 2. Подпись и создание AAB

### Автоматическая подпись (настроено в build.gradle.kts)
```bash
./gradlew bundleRelease
# Результат: app/build/outputs/bundle/release/app-release.aab
```

### Ручная проверка подписи
```bash
keytool -list -v -keystore release.keystore -alias release
```

## 3. Публикация в RuStore

### Шаг 1: Регистрация в RuStore
1. Перейдите на https://console.rustore.ru
2. Зарегистрируйте аккаунт разработчика
3. Заполните профиль разработчика

### Шаг 2: Создание приложения
1. В консоли нажмите "Добавить приложение"
2. Заполните информацию:
   - Название: "Генератор паролей"
   - Категория: Утилиты > Безопасность
   - Версия Android: 1.0.0
   - Минимальная SDK: 26
   - Целевая SDK: 35

### Шаг 3: Загрузка AAB
1. Загрузите app-release.aab
2. Укажите AdMob App ID (если используется)
3. Настройте цену (бесплатно или платно)

### Шаг 4: Настройка In-App Purchases
1. Перейдите в раздел "Покупки"
2. Создайте продукт:
   - ID: premium_99rub
   - Название: Премиум
   - Цена: 99 ₽
   - Тип: Необновляемая подписка

### Шаг 5: Загрузка政策性文件
1. privacy_policy.html - в разделе "Файлы"
2. Скриншоты экранов (минимум 3)
3. Иконку приложения

### Шаг 6: Отправка на модерацию
1. Проверьте все данные
2. Нажмите "Отправить на модерацию"
3. Ожидайте решение (обычно 1-3 дня)
